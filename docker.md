docker
======

## VM
- Build VMs:

**docker-machine _create | rm | start | stop | restart | kill_**

- VM status:

**docker-machine _active | config | env | ip | url | status | ls_**

- VM config:

**docker-machine _upgrade | regenerate-cert | provision_**

- VM interaction:

**docker-machine _ssh|scp_**

## Container
- List containers:

**docker _ps_**
