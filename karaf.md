karaf
======

- Install karaf docker
```
   docker 
   
   karaf is stored and accessible via ssh at karaf@karaf:
   ssh karaf@ip -p port 
   
   Install WebConsole:
   ssh login karaf:
      karaf@root()> feature:install webconsole
   By default, the Apache Karaf WebContainer port number is 8181.
```
